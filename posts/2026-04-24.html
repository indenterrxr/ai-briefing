AI NEWS 2026년 4월 24일

Google Cloud Next의 "에이전트 공장", SpaceX의 Cursor $60B 인수설, Moltbook 초대형 유출 사고

오늘의 키워드는 "에이전트 유통"과 "에이전트 보안 사고"입니다. Google이 라스베이거스에서 A2A 프로토콜과 Workspace Studio를 공개했고, SpaceX는 Cursor를 600억 달러에 인수한다는 소식으로 HN을 뒤흔들었으며, AI 에이전트 전용 SNS Moltbook의 DB가 그대로 열린 채 발견되었습니다.

1. Google Cloud Next 2026 — "모델 판매"에서 "에이전트 공장"으로

4월 22일 라스베이거스에서 열린 Google Cloud Next 2026에서 Google 클라우드 사업부는 기업 내부에서 AI 에이전트를 만들고 그 작업을 추적·관리할 수 있는 전면 개편된 도구 세트를 공개했습니다. 핵심 축은 세 가지입니다.

  • Workspace Studio — Docs·Sheets·Gmail에 직접 붙는 업무 에이전트를 노코드로 조립
  • A2A(Agent-to-Agent) 프로토콜 — 서로 다른 조직/벤더의 에이전트가 표준 포맷으로 협업
  • Deep Research Max — Gemini 3.1 Pro 기반, MCP 네이티브 통합과 자동 시각화로 장시간 자율 리서치 실행

Google의 메시지는 분명합니다. "모델만 파는 시대는 끝났고, 이제는 에이전트를 찍어내고 운영하는 플랫폼이 승부처"라는 선언으로, 지금까지의 OpenAI 정면 도전에서 → Anthropic과도 맞서는 풀스택 베팅으로 전선을 넓혔습니다.

2. SpaceX가 Cursor를 600억 달러에? — HN에서 731개 스레드로 폭발

SpaceX가 AI 코딩 어시스턴트 Cursor를 약 600억 달러에 인수한다는 보도가 나오면서 Hacker News에서만 731개 댓글이 달리는 난상토론이 벌어졌습니다. 쟁점은 크게 두 가지입니다.

  • 밸류에이션 적정성: 2025년 말까지 "AI 코딩 카테고리 리더"로 불리던 Cursor의 매출/성장률이 정말 이 가격을 정당화하는가
  • SpaceX의 전략적 의도: 위성·로켓 회사가 개발자 툴을 왜? — Starlink·Dojo·xAI와의 연계, 그리고 머스크 제국 내 "코드 생성 파이프라인" 확보설이 유력
업계에서는 "수직 통합의 새로운 임계점"이라는 평과 "AI 툴이 다시 한번 빅테크의 하드웨어 전쟁 카드가 됐다"는 분석이 동시에 나옵니다.

3. Moltbook 유출 — "AI 에이전트 SNS"에서 API 토큰 150만 건이 샜다

오늘 AI 보안 쪽에서 가장 충격적인 소식은 AI 에이전트 전용 소셜 네트워크 "Moltbook"의 DB가 인증 없이 완전히 열린 채 발견된 사건입니다.

  • 노출된 이메일 주소 약 3만 5천 건
  • 에이전트 API 토큰 약 150만 건 — 토큰 중 상당수는 여전히 유효 상태
  • 일부 비공개 메시지에는 평문 OpenAI API 키까지 포함

이번 사고는 "에이전트가 에이전트와 대화하는" 아키텍처의 비밀 관리(secret management)가 얼마나 취약한지를 드러내는 상징적 케이스로 빠르게 번지고 있습니다. 해당 플랫폼 사용 경험이 있는 조직은 API 키 전수 회전(rotation)을 권고합니다.

4. Google Antigravity(에이전틱 IDE) 원격 코드 실행 취약점

보안 리서처들이 Google의 에이전틱 IDE "Antigravity"에서 원격 코드 실행(RCE)으로 이어질 수 있는 취약점을 공개했습니다. 공격자가 조작된 프로젝트·프롬프트를 삽입해 개발자 로컬 환경에서 임의 코드를 실행시킬 수 있는 벡터로, "IDE가 스스로 코드를 쓰고 실행하는" 구조 자체가 새로운 공격면이라는 점을 다시 일깨운 사건입니다.

5. 미 연방법원 — "AI가 광고 콘텐츠에 '최종 권한'을 행사하면 플랫폼도 책임"

북부 캘리포니아 연방법원이 광고에 들어가는 콘텐츠 조립 과정에서 AI가 '최종 권한(ultimate authority)'을 행사하는 경우, 해당 플랫폼도 허위 진술의 "작성자(maker)"로 간주될 수 있다고 판시했습니다. 적용 대상은 Meta·Alphabet·Snap·TikTok·X Corp 등 주요 플랫폼 전반이며, 광고 생성/타깃팅에 AI를 적극 쓰는 모든 서비스에 중요한 판례입니다.

6. OpenAI GPT-5.5 — "도구를 갈아타며 혼자 일한다"

OpenAI가 GPT-5.5를 공개했습니다. 핵심 포지셔닝은 "여러 도구 사이를 자율적으로 갈아타며 복잡한 작업을 끝까지 수행하는 에이전트형 모델"로, 단일 호출로 웹·파일·코드 실행·데이터 조회를 조합합니다. Google의 A2A 발표와 맞물려, 이번 주는 "에이전트 아키텍처 경쟁"이 공식 개막한 주간으로 기록될 가능성이 큽니다.

7. ChatGPT Images 2.0 — 텍스트 렌더링·지시 준수 대폭 개선

OpenAI는 ChatGPT Images 2.0을 공개했습니다. 이전 모델에서 약점으로 지적되던 이미지 내 텍스트 렌더링 품질과 프롬프트 지시 준수 능력이 크게 개선됐고, 상위 플랜 사용자에게는 "thinking 기반 이미지 생성" 옵션까지 열렸습니다. Claude Design(디자인 산출물 협업)과 정면으로 부딪치는 기능군입니다.
