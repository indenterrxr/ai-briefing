1. OpenAI: ChatGPT의 다음 승부처는 더 오래 기억하는 인터페이스다

메모리 품질을 올리고 Free까지 넓힐 준비를 마쳤다

OpenAI의 Dreaming: Better memory for a more helpful ChatGPT 발표에 따르면, 회사는 ChatGPT의 메모리 시스템을 새 기반 위로 옮기고 있다. 핵심은 단순 저장이 아니라 과거 대화에서 관련 선호와 맥락을 더 정확하게 끌어오는 능력이다.

이번 업데이트에서 가장 중요한 숫자는 Free 사용자 제공 비용을 약 5배 줄였다는 점이다. OpenAI는 이 개선 덕분에 향후 몇 주 안에 Free 사용자에게도 dreaming 기반 메모리를 확대하고, Plus와 Pro의 메모리 용량도 늘릴 수 있게 됐다고 설명했다.

이는 최근 개인 금융 기능, 음성 dictation 개선과는 또 다른 층위의 변화다. OpenAI는 이제 새 모델 하나를 더 붙이는 것보다, 사용자 맥락을 오래 붙잡는 제품 레이어를 강화해 ChatGPT를 더 끈끈한 개인 비서로 만들려는 흐름을 보이고 있다.

모델 경쟁이 성능표에서 벌어진다면, 제품 경쟁은 기억력과 지속성에서 벌어진다.

2. Anthropic: 빌더 친화 정책을 꺼냈다가 곧바로 후퇴했다

Agent SDK 과금 규칙은 아직 확정본이 아니다

Anthropic의 지원 문서 Use the Claude Agent SDK with your Claude plan는 원래 2026년 6월 15일부터 Claude Agent SDK와 claude -p 사용량을 일반 구독 한도에서 분리하고, 별도 월간 크레딧을 주겠다고 안내했다.

하지만 같은 문서 상단에는 현재 해당 변경을 일시 중단했다는 업데이트가 붙어 있다. 즉, 지금은 아무것도 바뀌지 않았고, Agent SDK·claude -p·서드파티 앱 사용은 여전히 기존 구독 한도에 반영된다.

이 반전은 개발자 반응에서도 드러났다. 관련 Hacker News 글에는 39포인트가 붙었고, 별도 토론 글에서는 새 정책이 Max 요금제 가치를 흔든다는 불만이 빠르게 나왔다. Anthropic이 에이전트 생태계를 키우는 동시에, 전력 사용자 과금 구조를 어디까지 분리할지 아직 정교하게 못 박지 못했다는 신호로 읽힌다.

3. Google: Gemini는 이제 검색 다음으로 학교 안으로 들어간다

Classroom 문맥, 개인화 학습 노트, 무료 시험 대비까지 연결

Google은 ISTE 2026 교육 발표에서 Gemini 안에 새로운 Classroom 앱을 넣고, 교사가 실제 수업 문맥을 안전하게 활용해 업무를 돕도록 한다고 밝혔다.

여기서 끝나지 않는다. Google은 teacher-led AI activities를 Classroom에 확장하고, 학생용으로는 Study notebooks in Gemini를 출시해 개인별 학습 속도에 맞춘 수업과 퀴즈를 만들겠다고 했다. 또 ACT·GRE 대비 콘텐츠도 무료로 넓힐 계획이다.

이건 단순한 기능 묶음이 아니다. Google은 검색에서 Gemini를 기본화한 데 이어, 교육에서는 학교 데이터와 커리큘럼이 붙은 운영형 AI로 진입하려 한다. 사용 빈도가 높은 생산성 앱 안쪽에서 Gemini를 기본 도구로 굳히려는 전략이다.
