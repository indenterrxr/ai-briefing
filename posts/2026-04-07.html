오늘의 헤드라인

4월 첫째 주 AI 업계는 인프라 확장과 지정학적 공조가 가장 큰 화두였습니다. Anthropic이 Google·Broadcom과 멀티 기가와트급 TPU 계약을 체결했고, OpenAI·Anthropic·Google은 중국발 모델 복제(distillation) 시도에 공동 대응하기로 했습니다. 동시에 Google은 Gemini에 ChatGPT·Claude의 기억과 대화 이력을 옮겨오는 '임포트' 기능을 출시하며 이용자 락인 경쟁을 본격화했습니다.

1. Anthropic, Google·Broadcom과 멀티 기가와트 TPU 딜

Anthropic은 4월 7일 Google 및 Broadcom과의 파트너십을 확장해 차세대 TPU 수 기가와트급 용량을 확보했다고 발표했습니다. 신규 용량은 2027년부터 가동될 예정입니다.

  • Anthropic 연환산 매출(ARR)이 2025년 말 90억 달러에서 300억 달러 이상으로 급증
  • 포스트머니 기업가치 약 3,800억 달러로 평가
  • Broadcom은 같은 날 Google과의 칩 공급 계약도 확장
이번 딜은 GPU 일변도의 AI 인프라 시장에서 커스텀 ASIC(TPU 계열)의 비중이 빠르게 커지고 있음을 보여주는 신호입니다.

2. 미국 빅3, 중국 모델 복제에 공동 대응

OpenAI·Anthropic·Google이 Frontier Model Forum을 통해 정보를 공유하며, 중국 경쟁사들이 약관을 위반해 최첨단 미국 모델의 출력을 추출(adversarial distillation)하는 시도를 차단하기 시작했습니다. 세 회사가 경쟁 관계임에도 공동 전선을 펴는 것은 이례적입니다.

Anthropic 측은 내부적으로 "Claude Mythos"로 불리는 프런티어 모델을 테스트 중이며, "능력의 단계적 도약(step change)"이라는 표현이 나오고 있습니다.

4. Gemini, ChatGPT·Claude 기억 가져오기 출시

Google은 Gemini에 Import Memory 및 Import Chat History 기능을 추가했습니다. 사용자가 ChatGPT나 Claude에 축적해 둔 개인화 정보와 대화 기록을 그대로 옮겨올 수 있게 한 것으로, AI 어시스턴트 시장에서 본격적인 전환 비용 낮추기 경쟁이 시작됐다는 평가입니다.

6. [단독 주목] Claude Mythos, 제한 배포 — "너무 강력해서 공개 불가"

오늘 GeekNews에서 가장 뜨거웠던 이슈. Anthropic이 신규 모델 Claude Mythos를 일반에 공개하지 않고, "Project Glasswing"이라는 제한 프리뷰를 통해 선별된 보안 연구 파트너에게만 배포하기로 결정했습니다. 이유는 단순합니다 — 사이버 보안 취약점 탐지 능력이 너무 강력해서 악용 시 전 세계 인프라에 위협이 될 수 있다는 판단입니다.

Mythos의 실제 능력

  • 주요 OS·웹브라우저에서 수천 건의 고위험 취약점을 이미 발견
  • Firefox 147 JS 엔진 실험: Opus 4.6은 수백 회 중 2회 성공, Mythos는 181회 성공 + 레지스터 제어 29회 추가 성공
  • 브라우저 익스플로잇에서 취약점 4개를 연쇄 활용해 렌더러·OS 샌드박스 모두 탈출하는 공격 코드 작성
  • Nicholas Carlini: "OpenBSD에서 27년간 존재해 온 크래시 유발 버그를 찾았다" — 해당 버그는 2026년 3월 25일 패치 완료

Project Glasswing

Anthropic은 AWS·Apple·Microsoft·Google·Linux Foundation 등을 파트너로 참여시켜 각자 시스템의 취약점을 먼저 찾아 수정할 수 있게 했고, 1억 달러 상당의 사용 크레딧과 400만 달러의 오픈소스 보안 단체 기부도 함께 공개했습니다. Simon Willison은 "'우리 모델이 너무 위험하다'가 보통은 마케팅이지만 이번엔 신중함이 타당하다"며 제한 배포를 지지했습니다.

시사점: AI의 취약점 탐지 능력이 이미 인간 전문가를 압도하는 수준에 도달했다는 것. 방어 측에는 기회이자, 공격 측 손에 들어갈 경우 AI가 가져올 가장 현실적인 위험 중 하나입니다.
